# Изучение Python 🐍

Добро пожаловать в мой путь изучения Python! Здесь я буду собирать заметки, примеры и мини-проекты

---

## Beginner

- **Ввод/вывод, переменные**: `print()`, `input()`, `sep`, `end`, PEP8  
- **Арифметика**: int, float  
- **Условные конструкции**: `if/else`, вложенные условия, логические операции  
- **Циклы**: `for`, `while`, `range()`, `break/continue/else`, вложенные циклы  
- **Строки**: индексация, срезы, методы, форматирование, Unicode, сравнение  
- **Списки**: основы, методы, `split()/join()`, списочные выражения, сортировка  
- **Функции**: без параметров, с параметрами, локальные/глобальные переменные, возврат значения  
- **Мини-проекты**: модуль `random`  

---

## Advanced

- Типы: `bool`, `NoneType`  
- Вложенные списки и матрицы  
- Кортежи (`tuple`), множества (`set`), словари (`dict`)  
- Модуль `random`, работа со строками (`string`)  
- Типы данных: `decimal`, `Fraction`, `complex`  
- Графика: модуль `turtle`  
- Дополнительные возможности функций  
- Работа с текстовыми файлами
